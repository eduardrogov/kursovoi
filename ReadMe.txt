RGBtoYUVandVV
Обзор проекта

RGBtoYUVandVV.vcxproj
	Это основной файл проекта, сгенерированный с помощью компилятора.
	Он содержит информацию о версии Visual C++, которая сгенерировала файл и иинформация о платформах, конфигурациях и возможностях проекта, собранном в компиляторе. 

RGBtoYUVandVV.vcxproj.filters
	Это файл фильтров для проектов VC++, сгенерированный с помощью мастера приложений.
	Он содержит информацию об ассоциации между файлами в проекте и фильтры. Эта ассоциация используется в среде IDE для отображения группировки файлов с аналогичных расширений под определенным узлом (например, “.cpp” файлы связаны с фильтром «Исходные файлы»)

RGBtoYUVandVV.h
	Это заголовочный файл для приложения. Он включает другие заголовки (включая Resource.h) и объявляет класс CRGBtoTUVandVVApp

RGBtoYUVandVV.cpp
	Это основной исходный файл приложения, который содержит в себе непосредственно класс CRGBtoTUVandVVApp

RGBtoYUVandVV.rc 
	Это список всех ресурсов Microsoft Windows, которые программа использует. Он содержит значки, растровые изображения и курсоры, которые хранятся в каталоге RES. Этот файл можно напрямую редактировать в Microsoft Visual C++. 

res\RGBtoYUVandVV.ico
	Это файл значков, который используется для значка нашего приложения. Этот значок включен в основной файл ресурсов RGBtoYUVandVV.rc

Res\RGBtoYUVandVV.rc2
	Этот файл содержит ресурсы, которые не редактируются Microsoft Visual C++.
/////////////////////////////////
Мастер приложений создает один класс диалога: 
	RGBtoYUVandVVDlg.h, RGBtoYUVandVVDlg.cpp - диалог
    Эти файлы содержат ваш CRGBtoYUVandVVDlg. Этот класс определяет поведение основного диалога приложения. Шаблон диалогового окна в RGBtoYUVandVV.rc, который можно редактировать в Microsoft Visual C ++.
/////////////////////
Другие преимущества: 
	Элементы управления ActiveX
	Приложение включает поддержку использования элементов управления ActiveX.

///////
Другие стандартные файлы:

StdAfx.h, StdAfx.cpp
	Эти файлы используются для создания файла с предварительно скомпилированным заголовком (PCH) и именем RGBtoYUVandVV.pch и файл StdAfx.obj

Resource.h
	Это стандартный заголовочный файл, который определяет новые идентификаторы ресурсов. Microsoft Visual C++ читает и обновляет этот файл. 

RGBtoYUVandVV.manifest
Файлы манифеста приложений используются Windows XP для описания зависимости приложения от конкретных версий сборок. Этот загрузчик использует эту информацию для загрузки соответствующей сборки из кеша сборки или частного из приложения. Явление приложения может быть включено для перераспределения как внешний манифестный файл, который установлен в той же папке, что и приложение или он может быть включен в исполняемый файл в виде ресурса.
////////////////////////////////////////////////// ///////////////////////////

Другие примечания:
Мастер приложений использует «TODO:», чтобы указать части исходного кода, который необходимо добавить или настроить. Если приложение использует MFC в общей DLL, необходимо будет перераспределить библиотеки MFC. Если приложение находится на языке, отличном от локального операционной системы, также придется перераспределять соответствующие локализованные ресурсы MFC100XXX.DLL.
